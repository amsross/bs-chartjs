## Installation

Install the project:

`npm install --save @amsross/bs-chartjs`

And add the dependency to your bs-dependencies in `bsconfig.json`:

```json
"bs-dependencies": [
  "@amsross/bs-chartjs"
]
```

The project will be available under the `chartjs` namespace
